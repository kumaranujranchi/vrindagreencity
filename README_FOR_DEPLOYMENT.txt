IMPORTANT: UPLOAD INSTRUCTIONS

1. Upload 'index.php' to your public_html folder.
   - This file replaces your existing 'index.html'.
   - You can delete or rename your old 'index.html' to 'index_backup.html'.

2. Upload the 'inc' folder.
   - It contains 'push-notification-widget.php'.

3. Upload the 'assets' folder.
   - It contains 'js/push-notifications.js'.

4. Upload 'service-worker.js' to the root folder (public_html).

5. Upload the 'admin' folder (if you haven't already).

Once uploaded, visit your website. You should see the permission popup automatically!
